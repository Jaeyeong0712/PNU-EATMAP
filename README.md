프로젝트 소개: 부산대학교 맛집 지도
카카오맵에서 부산대학교 주변 음식점 정보를 자동으로 수집하고, Folium 기반의 인터랙티브 지도로 시각화하는 프로젝트입니다.
Streamlit을 활용해 누구나 쉽게 사용할 수 있는 웹 앱 형태로 구현되었으며, 맛집 탐색·필터링·통계 분석 기능을 제공합니다.

프로젝트 팀원: 202641124 김재영, 202641130 김태은

실행방법: app.py를 다운받은 뒤 Streamlit으로 실행한다.

사용 라이브러리:
streamlit, streamlit-folium, folium, selenium, webdriver-manager, beautifulsoup4, requests, lxml, pandas, numpy, python-dotenv
