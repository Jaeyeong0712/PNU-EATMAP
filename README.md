프로젝트 소개: 부산대학교 맛집 지도
카카오맵에서 부산대학교 주변 음식점 정보를 자동으로 수집하고, Folium 기반의 인터랙티브 지도로 시각화하는 프로젝트입니다.
Streamlit을 활용해 누구나 쉽게 사용할 수 있는 웹 앱 형태로 구현되었으며, 맛집 탐색·필터링·통계 분석 기능을 제공합니다.

프로젝트 팀원: 202641124 김재영, 202641130 김태은

실행방법:
1. 라이브러리 설치 : pip install -r requirements.txt

2. 프로그램 실행 : streamlit run app.py

3. 브라우저 접속

사용 라이브러리:
streamlit
streamlit-folium
folium
requests
pandas

주의사항
크롤링 시 해당 서비스의 이용약관을 확인하고 사용해주세요.
과도한 요청은 IP 차단의 원인이 될 수 있습니다.
카카오맵 API 사용 시 카카오 개발자 센터에서 REST API 키를 발급받아야 합니다.
